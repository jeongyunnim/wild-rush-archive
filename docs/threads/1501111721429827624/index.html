합주 리뷰 채널 개설 및 목적

  • · 합주 진행 후 리뷰를 남기는 포럼 채널로 운영
  • · 당일 저녁 합주 요약 봇이 양식에 따라 리뷰를 파싱 후 정리

발언: 령

리뷰 작성 양식 규정

  • · 반드시 양식을 복사하여 리뷰를 작성해야 함
  • · 제목 형식: 🎸 합주 리뷰 - [날짜]
  • · 1. 곡명 / 2. 곡명 형태로 곡 리스트 작성
  • · 총평과 추가 내용 섹션 포함

발언: 령

곡 리뷰 작성 가이드라인

  • · 모든 곡을 리뷰할 필요 없음, 리뷰할 내용이 있는 곡만 추가
  • · "굿", "좋아요" 같은 단순 리뷰는 지양

  • · 다음 합주는 곡 순서를 바꿔 진행 희망

# 합주 # 선곡

합주 진행 후 리뷰를 남기는 포럼 채널 입니다. ⚠️**각 날짜에 맞는 포스트에 아래 양식을 지켜 리뷰를 남겨주세요.**⚠️ 남겨 주신 리뷰는 당일 저녁 🤖합주 요약 봇🤖에 의해 정리 됩니다. 합주 요약 봇은 아래 양식에 따라 각 리뷰를 파싱 후 정리합니다. 따라서 !꼭! 양식을 카피하여 리뷰를 작성해주세요. ======================================================== ** 양식 ** ``` :guitar: 합주 리뷰 - 1. get your gun 2. summer -------------------------------------- :hand_splayed: 총평 --------------------------------------- :dove: 추가 내용 ``` ⚠️ 곡 리뷰 모든 곡을 리뷰 할 필요는 없습니다. 리뷰 할 내용이 있는 곡만 리스트에 추가하여 리뷰 부탁 드립니다. "굿", "좋아요" 와 같은 리뷰는 지양합니다. (리뷰 요약 시 내용이 불필요하게 길어집니다.) ⚠️ 총평 총평은 모든 멤버 필수 입니다. 해당 섹션은 "굿", "좋아요" 와 같은 리뷰도 가능합니다. 하지만, 가급적 구체적이고 도움이 될 만한 다양한 리뷰를 지향합니다. ⚠️ 추가 내용 해당 섹션은 곡 리뷰, 총평이 아닌 다른 의견을 남기는 부분입니다. 합주 때 나온, 회식 때 나온 다양한 의견을 남겨주세요.
