**📢 26년 하반기 공연 공지 📢** ﻿ 와일드러시는 26년 하반기 공연을 목표 하고 있습니다. 해당 공연은 다른 팀과 함께 하는 공연이 아닌, **와일드러시 단독 공연**입니다. 곡 수 : 총 15곡 (팀별 5곡) 공연 날짜 : 미정 (8월 예상) 공연 위치 : 미정 (홍대 예상) .

각자 최대 3곡 추천, 본인 노래 투표 불가, 시트 미선택 곡은 합주/공연 불가, 커버곡은 악보 확인 필수. 곡 선정 시트 링크 공유.

**📢 선곡 추천 공지 사항 📢** 선곡 추천 리스트 ---> [선곡 리스트 시트](https://docs.google.com/spreadsheets/d/115cwrD2uFfh-KSpgiQGBr9DJoa0hfeqBdLgdQK4Hxp0/edit?pli=1&gid=1608245828#gid=1608245828) **곡 추천 Ground Rules:** 1. 각자 최대 3곡까지 추천 가능 (0곡 ~ 3곡) 2. 추천자는 본인 노래 투표 불가능 3. 해당 시트에서 선택되지 않은 곡은 합주 및 공연 불가능 4. 커버곡의 경우 악보 유무 확인 필수 (악보 없는 경우 세션 의견 우선 반영) 5. 선곡 리스트 시트에 곡 업로드 후 각 선곡 채널(ex. 선곡-인호)에 추천곡 링크 업로드. **최종 곡 선정 방식 (2 Step)** 1차 - 팀 별로 회의하여 여섯곡 결정 (5곡 : 4곡 + 엔딩 1곡 / 1곡 : 예비곡) 2차 - 멤버 전체 투표 후 최종 결정 (15곡). 곡 회의는 팀별로 진행하되, 선택한 곡은 모든 멤버의 컨펌 후 최종 곡으로 결정될 수 있습니다. * 전체 멤버 컨펌: 해당 과정은 팀별로 선택한 곡이 전체 공연 분위기에 맞는지 판단하기 위한 과정입니다. 곡 난이도, 각자의 취향을 반영하여 곡 선택을 하는 과정이 아닙니다. 따라서 모든 멤버가 곡 진행 상황을 확인할 수 있도록 추천곡 리스트는 [선곡 리스트 시트](https://docs.google.com/spreadsheets/d/115cwrD2uFfh-KSpgiQGBr9DJoa0hfeqBdLgdQK4Hxp0/edit?pli=1&gid=1608245828#gid=1608245828) 에 업데이트 해주시고, 해당 시트에서 1차 투표를 통해 과반의 동의를 얻은 곡만 회의 진행할 수 있도록 부탁드립니다. .
