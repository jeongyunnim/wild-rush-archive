# Wild Rush Archive — 페이지 재설계

## 현재 상태

| 페이지 | 기능 | 한계 |
|--------|------|------|
| index.html | 채널 목록 + 메시지 수 | 내용 파악 불가 — 클릭해야 알 수 있음 |
| channel.html | 채널 내 메시지 나열 + 태그 | 태그만 있고 요약 없음, 스크롤 피로 |
| thread.html | 스레드 메시지 + 요약 | 스레드 단위만, 채널 전체 맥락 없음 |
| tag_*.html | 태그별 메시지 모음 | 태그 자체가 fragmentary |

**핵심 문제**: "이 채널에서 무슨 이야기가 오갔는지" 한눈에 파악 불가

---

## 설계 목표

1. **채널별 토픽 요약** — 채널 진입 시 "이 채널의 주요 주제 5가지" 즉시 노출
2. **토픽 → 메시지 드릴다운** — 주제 클릭 시 관련 메시지만 필터링
3. **결정사항 하이라이트** — 투표 결과, 확정 일정, 선정 곡 등 핵심 결론 상단 노출
4. **타임라인 + 토픽 듀얼 뷰** — 시간순 vs 주제별 전환 가능

## 구현 계획

### Phase 1: 데이터 파이프라인 확장

| 작업 | 파일 | 설명 |
|------|------|------|
| 1-1 | `src/topic_extractor.py` (신규) | 채널 메시지를 토픽으로 클러스터링 (LLM) |
| 1-2 | `src/channel_summarizer.py` (신규) | 채널 전체 요약 + 결정사항 추출 |
| 1-3 | `crawler.py` 수정 | 크롤링 후 topic_extractor, channel_summarizer 호출 |
| 1-4 | `guild_data.json` 스키마 확장 | `channel_summaries` 필드 추가 |

### Phase 2: 템플릿 재작성

| 작업 | 파일 | 설명 |
|------|------|------|
| 2-1 | `templates/index.html` | 대시보드 스타일로 전면 개편 |
| 2-2 | `templates/channel.html` | 토픽 중심 레이아웃 + 시간순/주제별 토글 |
| 2-3 | `templates/topic.html` (신규) | 토픽 상세 페이지 |
| 2-4 | `renderer.py` 수정 | 신규 페이지 렌더링 추가 |

### Phase 3: UX 개선

| 작업 | 설명 |
|------|------|
| 3-1 | 검색 기능 (클라이언트 사이드 JS) |
| 3-2 | 토픽 필터 + 날짜 필터 |
| 3-3 | 모바일 최적화 |

## 토픽 추출 프롬프트 (예시)

```
You are a topic extractor for a Korean band community Discord archive.
Given all messages in a channel, group them into coherent topics.

Output format — ONLY valid JSON:
{
  "topics": [
    {
      "name": "토픽 제목 (5자 이내)",
      "summary": "토픽 요약 (2문장 이내)",
      "decision": "결정사항 (있으면). 없으면 null",
      "message_ids": ["msg_id_1", "msg_id_2", ...],
      "importance": "high" | "medium" | "low"
    }
  ],
  "key_decisions": ["전체 채널 핵심 결정 1", "..."],
  "important_links": [{"url": "...", "title": "..."}]
}

Rules:
- 최대 10개 토픽
- 단순 리액션/인사 메시지는 제외
- 결정사항은 "확정", "결정", "선정" 등 확실한 결론만
- importance: 공연/선곡/일정 관련은 high, 잡담은 low
```

## 예상 결과

| Before | After |
|--------|-------|
| 채널 클릭 → 메시지 스크롤 → 태그 확인 | 채널 카드에서 주요 주제 즉시 확인 |
| "공연 언제더라?" → 전체 검색 | 홈에서 "최근 결정사항" 확인 |
| 태그별 페이지는 fragmentary | 토픽 페이지에서 맥락 있는 요약 |
